# smart-home-energy-optimization-ai
An AI-powered dashboard for Kerala residents using XGBoost and Llama 3 to predict KSEB bills. Features real-time location/weather sync, 2026 telescopic billing math, and bilingual (English/Malayalam) AI tips. Optimize home energy, avoid peak-hour penalties, and manage appliance costs with precision-clamped ML models. Requires Ollama for AI.
